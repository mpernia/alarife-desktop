Nueva Enciclopedia
INICIO - Enciclopedia
Describe cómo se crea una nueva enciclopedia.


Contenido:

  • Nombres.
  • Resumen y Plantilla.
  • Usuario.
  • Tipos de archivos.
  • Finalizar.

Para acceder a esta opción usted puede hacerlo por las siguientes vías: 1- Barra de Menú → Enciclopedia → Nuevo. 2- Pulsar la combinación de teclas Ctrl + N. 3- Barra de herramientas hacer clic en el ícono

Nombres.

Primer paso del asistente para crear enciclopedia. En esta sección del asistente para la creación de un nuevo proyecto de enciclopedia, se escribe el título de la enciclopedia y el nombre; el nombre seleccionado es utilizado por el software para crear en el directorio que almacena todos los proyectos un fichero con extensión *.alf y un directorio.

Resumen y Plantilla.

Segundo paso del asistente para crear enciclopedia. En esta sección usted puede escribir un resumen sobre el contenido de su proyecto de enciclopedia.

Ventana para seleccionar la plantilla para la enciclopedia. Para seleccionar una plantilla haga clic en la vista en miniatura que representa el diseño de las plantillas.

Usuario.

Tercer paso del asistente para crear enciclopedia. Área para la creación del Editor Jefe de la nueva enciclopedia, vea el apartado de tipos de usuario para más información del rol de este usuario.

Tipos de archivos.

Cuarto paso del asistente para crear enciclopedia. Área destinada para que usted seleccione los tipos de extensiones de archivos que serán gestionados en su proyecto.

Finalizar.

Quinto paso del asistente para crear enciclopedia. En este apartado se muestra un resumen de las principales características que tendrá su proyecto, de no ser de su agrado puede utilizar el botón "Anterior" y volver a seleccionar los datos que mejor satisfagan sus necesidades. Luego de hacer clic en el botón "Finalizar" Alarife 4 mostrará la ventana principal con un dialogo de verificación de usuario, para comenzar a editar el contenido de la enciclopedia creada. Es válido señalar que en esta verificación de usuario aparecen dos usuarios: a) El Súper usuario. b) El usuario creado en el paso tres del asistente.
