Este espacio de trabajo le permite agregar nuevos usuarios a sus enciclopedias. Puede acceder a esta opción desde: - Barra de menú → Ayuda → Preferencias. - Barra de herramienta clic en el icono.
 
Interfaz para la gestión de usuarios
 
Tipos de usuarios.
 
Alarife IV gestiona tres tipos:
 
Usuarios Descripción
Súper Usuario Usuario que controla todas las opciones del programa, además de tener acceso total a toda la información en la enciclopedia que está en edición.
Editor Jefe Usuario con la responsabilidad de dirigir y fiscalizar el desarrollo de una enciclopedia.
Editor Usuario que interactúa con el software para crear el contenido de las enciclopedias.
 
Privilegios de usuarios.
 
Usuarios Privilegios
Súper Usuario Acceso total a las opciones del software y a la enciclopedia en edición.
Editor Jefe Insertar, eliminar y modificar todos los datos recopilados en la enciclopedia.
Editor Insertar, eliminar y modificar los artículos que ha creado. Insertar Medias y Actores. Editar descripción de Medias y Actores. Visualizar todo el contenido de la enciclopedia.
 
Gestionar usuarios.

Alarife de forma predeterminada gestiona dos tipos de usuario "Súper Usuario" y "Editor Jefe". Si usted va a trabajar en colectivo es aconsejable que se active la opción de "Gestión de usuarios" así podrá establecer el acceso a cada miembro de su equipo de edición.

  • Súper usuario:

Este tipo de usuario es único, y solo se puede cambiar su contraseña y se recomienda hacerlo pues como se detalla en otro apartado tiene una contraseña predeterminada y pública, siendo una posible vulnerabilidad en cuanto a seguridad se refiere. Para hacer efectivo el cambio de contraseña hacer clic en el botón "Cambiar contraseña".

  • Insertar usuario:

Para insertar un usuario haga clic en el botón "Insertar". Aparecerá el diálogo para la inserción de usuarios en este Ud. debe escribir: - Nombre completo del usuario. -Usuario. -Contraseña. -Tipo de usuario (Editor Jefe, Editor).

Diálogo para Insertar usuarios.

  • Editar usuario:

Para editar un usuario Ud. puede hacerlo por dos vías: 1- Doble clic sobre el nombre del usuario a editar. 2- Clic sobre el botón "Editar", para editar el usuario activo. Luego puede cambiar los datos que sean necesarios.

Diálogo para editar usuario.

  • Eliminar usuario:

​Para eliminar un usuario solo debe hacer clic sobre el botón "Eliminar" y se eliminará el usuario activo en la lista de usuarios.
