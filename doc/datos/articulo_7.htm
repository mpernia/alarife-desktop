INICIO - Enciclopedia
Describe como abrir una enciclopedia creada con anterioiridad


Usted puede abrir una enciclopedia por las siguientes vías:

- Haciendo doble clic sobre un fichero *.alf.
- Barra de menú → Enciclopedia → Abrir. 
- Barra de herramientas, clic sobre el icono. 
- Desde una línea de comandos, escriba este comando y pulse la tecla Intro:: alarife ruta_mi_fichero.alf 

Nota: el primer modo o vía en escritorios GNU/Linux como Gnome o KDE debe configurarse de forma manual. En Windows no es necesario.
