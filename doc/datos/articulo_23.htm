Formas para acceder a Alarife 4:

  • Microsoft Windows XP:
  • Inicio → Programas → Alarife 4 → Alarife
  • Opción de Ejecutar Programas (WIN + R) escribir "alarife"
  • Abrir un fichero *.alf
  • Desde la ventana de comandos de DOS "CMD" escribir "alarife"
  • GNU/Linux:
  • Menú → Programación → Alarife 4
  • Menú → Internet → Alarife 4
  • Opción de Ejecutar Programas (Alt + F2), escribir "alarife"
  • Abrir un fichero *.alf
  • Desde un shell de comandos escribir "alarife"

Opciones predeterminadas:

  • La contraseña del Super Usuario es "alarife" pero siempre usted podrá cambiarla. Para más información ver el contenido relacionado con la gestión de usuarios.
  • Cada vez que se importa una enciclopedia nueva, se crea un usuario con nombre "Editor Jefe" y contraseña "alarife", el nombre y la contraseña de este usuario usted puede cambiarlos.Para más información ver el contenido relacionado con la gestión de usuarios.
  • Todas las enciclopedias que se crean o se importen se almacenarán en un directorio que crea el programa con nombre "Alarife", en las siguientes rutas:
  • En Microsoft Windows XP "C:/Documents and Settings/Alarife"
  • En GNU/Linux "/home/user/Alarife"

Áreas o secciones en las que se divide Alarife 4:

1- Barra de Menú y Barras de Herramientas: en esta área estan todos los vínculos hacia las diferentes acciones que Ud. puede realizar con el software.

  • Estandard: nuevo, abrir, estadísticas, propiedades, configuración, autores y medias.
  • Edición: agregar tema, editar tema, eliminar tema, ordenar temas, agregar artículo, editar artículo, eliminar artículo, mover artículo y buscar.
  • Herramientas: importar enciclopedia, exportar enciclopedia y generar.
  • Ayuda: manual de usuario.

2- Listado de Temas: muestra el listado de los temas registrados en la enciclopedia. Ir al Inicio: esta opción le permite volver a los primeros temas. Ir Atrás: esta opción le permite retroceder un nivel en el árbol de temas.

3- Listados de Artículos: muestra el listado de los artículos relacionados con el tema actual.
